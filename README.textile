ちょこっと便利なExcelマクロを作っていこうと思います。
Excel2010 で作成・動作確認しています。

# BookList
** 蔵書リストのシートとマクロ。ISBNを入力してマクロを実行するとAmazonからタイトル等を取得して書き出します。
** Amazon.comのアクセスキー、アソシエイト・タグが必要です。
** 2007以前のExcelでもたぶん動きます。(basをインポートしてください)
